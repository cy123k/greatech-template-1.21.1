# Art Direction

## Tooling

Recommended tools:

- `Blockbench` for model work
- `Photoshop` for texture work

Optional:

- `Blender` for concept exploration only

## Visual Direction

The target style is:

- `Create` mechanical structure
- `GregTech` industrial electrical detailing

This means the machine should not feel purely wooden/brass and not feel like a plain flat tech cube either.

## Visual Keywords

- industrial
- electromechanical
- generator
- vented casing
- steel
- brass
- copper
- coil
- flywheel
- control panel
- hazard stripe

## SU Energy Converter Shape Direction

Recommended composition:

- dark steel machine casing
- visible rotational generator core
- brass mechanical shaft input ring
- electric output face with panel details
- vents, bolts, and segmented plating

## Suggested Face Roles

- front: status panel or viewing window
- back: shaft input side
- left: vents or coil casing
- right: electric connector details
- top: service panel and warning stripe
- bottom: simple structural metal base

## Animation Direction

First animation target:

- one visible rotor or generator drum that spins with the machine

Do not overbuild the first pass. One strong moving element is enough.

## Texture Direction

Recommended starting resolution:

- `32x32`

Suggested palette:

- dark steel gray
- iron gray
- brass / copper accents
- teal or cyan electric accent
- small yellow-black hazard markers

## Production Order

1. rough silhouette in Blockbench
2. face breakdown by function
3. quick base texture pass
4. spinning rotor part separation
5. in-game test
6. final polish pass
